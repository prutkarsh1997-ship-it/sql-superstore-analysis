# sql-superstore-analysis
SQL project analyzing sales, profit, and business insights using Superstore dataset

Superstore Sales Analysis (SQL Project)

##  Overview
This project analyzes sales and profit data using SQL to derive meaningful business insights.

##  Key Analysis Performed
- Total Sales and Profit calculation
- Top products by sales
- Region-wise and state-wise performance
- Customer analysis (top customers by sales and profit)
- Category and sub-category performance
- Time series analysis (year-wise and monthly trends)
- Identification of loss-making products
- Profit margin analysis

##  Key Insights
- Technology category has the highest profit margin (~17%)
- Furniture has high sales but very low profitability (~2.5%)
- Some products generate sales but result in losses
- New York is more profitable compared to other states

##  Skills Used
- SQL (Aggregation, GROUP BY, HAVING, ORDER BY, YEAR(), MONTH(), ROUND())
- Data Analysis
- Business Insight Generation

##  Dataset
Superstore dataset (Kaggle)
